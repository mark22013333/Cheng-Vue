## 技術亮點

### 1. 非同步任務架構
```
掃描 ISBN → 建立任務（返回 taskId） → 立即返回
                ↓
         後端 @Async 執行爬取
                ↓
         前端輪詢查詢狀態（2 秒間隔）
                ↓
         完成後彈出通知
```

### 2. 連續掃描體驗
- 掃描後相機不關閉，可連續掃描多個 ISBN
- 每個任務獨立處理，互不干擾
- 任務數量即時顯示，進度一目了然

### 3. 智能降級方案
```
FlareSolver 失敗
    ↓
books.com.tw 搜尋書名
    ↓
智能匹配 alt 屬性
    ↓
提取真實圖片 URL
    ↓
下載成功
```

### 4. 拖動 + 新手導航
- 拖動功能：觸控事件處理 + 邊界限制 + 位置記憶
- 新手導航：灰暗遮罩 + 高亮動畫 + 箭頭指引 + 提示文字

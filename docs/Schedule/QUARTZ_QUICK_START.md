# 定時任務快速使用指南

## 📋 功能概述

現在您可以透過友善的介面建立定時任務，不需要手動輸入 Bean 名稱和方法名稱！

---

## 🚀 使用步驟

### 1. 進入定時任務頁面

```
系統管理 → 系統監控 → 定時任務
路徑: /monitor/job
```

### 2. 點選「新增」按鈕

會彈出新增任務對話框

### 3. 選擇設定方式

您有兩種方式可以選擇：

#### 方式 A：從範本選擇（推薦）

```
1. 選擇「從範本選擇」
2. 從下拉選單選擇任務類型（如：執行爬蟲（無參數））
3. 系統自動填入 Bean 名稱和方法名稱
4. 根據提示填寫參數（有預設值範例）
5. 可使用建議的 Cron 表達式
6. 儲存
```

**優點**：
- ✅ 不會拼錯 Bean 或方法名稱
- ✅ 參數有清楚說明和範例
- ✅ 自動產生呼叫目標字串
- ✅ 提供 Cron 表達式建議

#### 方式 B：手動輸入

```
1. 選擇「手動輸入」
2. 手動輸入呼叫目標字串
3. 輸入 Cron 表達式
4. 儲存
```

**適用場景**：
- 進階使用者
- 需要客製化的呼叫方式

---

## 📝 實際範例

### 範例 1：建立每天執行的 CA102 爬蟲

**步驟**：
1. 點選「新增」
2. 任務名稱：`CA102 台股重大訊息爬蟲`
3. 任務分組：`DEFAULT`
4. 設定方式：選擇「從範本選擇」
5. 任務類型：選擇「執行爬蟲（無參數）」
6. 爬蟲類型：輸入 `CA102`（已預填）
7. Cron 表達式：`0 0 1 * * ?`（已預填，每天凌晨1點）
8. 點選「確定」

**結果**：
- 呼叫目標：`crawlerTask.run('CA102')`
- 每天凌晨 1 點執行

### 範例 2：建立多次執行的爬蟲（只爬今日）

**步驟**：
1. 點選「新增」
2. 任務名稱：`CA102 今日訊息爬蟲`
3. 任務分組：`DEFAULT`
4. 設定方式：選擇「從範本選擇」
5. 任務類型：選擇「執行爬蟲（帶模式）」
6. 參數填寫：
   - 爬蟲類型：`CA102`
   - 執行模式：`today-only`
7. Cron 表達式：`0 0 9,12,15 * * ?`（已預填，每天9、12、15點）
8. 點選「確定」

**結果**：
- 呼叫目標：`crawlerTask.runWithMode('CA102', 'today-only')`
- 每天 9、12、15 點執行

### 範例 3：建立完整參數的爬蟲

**步驟**：
1. 點選「新增」
2. 任務名稱：`CA102 完整同步`
3. 任務分組：`DEFAULT`
4. 設定方式：選擇「從範本選擇」
5. 任務類型：選擇「執行爬蟲（完整參數）」
6. 參數填寫：
   - 爬蟲類型：`CA102`
   - 執行模式：`full-sync`
   - 批次大小：`1000`
   - 逾時時間(ms)：`120000`
7. Cron 表達式：`0 0 2 * * ?`（已預填，每天凌晨2點）
8. 點選「確定」

**結果**：
- 呼叫目標：`crawlerTask.runAdvanced('CA102', 'full-sync', 1000, 120000)`
- 每天凌晨 2 點執行

## 🎯 可用的任務類型

### 爬蟲任務

| 任務類型 | 說明 | 適用場景 |
|---------|------|---------|
| **執行爬蟲（無參數）** | 使用預設參數執行 | 簡單執行，每日例行爬取 |
| **執行爬蟲（帶模式）** | 可設定執行模式 | 需要控制爬取範圍（如只爬今日） |
| **執行爬蟲（完整參數）** | 支援批次大小、逾時等設定 | 需要效能調整 |
| **執行爬蟲（日期範圍）** | 指定開始和結束日期 | 補爬歷史資料 |
| **執行爬蟲（所有參數）** | 最完整的參數配置 | 複雜場景，完全客製化 |

## 💡 使用技巧

### 1. Cron 表達式建議

當您選擇任務類型後，系統會自動建議常用的 Cron 表達式：

- 點選「使用建議值」按鈕可直接套用
- 或點選「產生表達式」使用圖形化工具

### 2. 參數預設值

每個參數都有範例值，可以：
- 直接使用預設值
- 或修改為您需要的值

### 3. 必填參數提示

- 必填參數會顯示 ⚠️ 標記
- 提交時會自動驗證

### 4. 呼叫目標預覽

在範本模式下，系統會即時顯示產生的呼叫目標字串，讓您確認無誤。

## 🔧 常見問題

### Q1: 如何切換不同的爬蟲類型？

**A**: 在「爬蟲類型」參數中修改為其他類型即可：
- `CA102` - 台股重大訊息
- `CA103` - （未來新增的其他爬蟲）
- ...

### Q2: 如何立即執行測試？

**A**: 儲存任務後：
1. 在列表中找到該任務
2. 點選「執行」按鈕
3. 系統會立即執行一次

### Q3: 如何查看執行記錄？

**A**: 點選頁面上方的「日誌」按鈕，可查看所有任務的執行記錄。

### Q4: 手動模式和範本模式可以切換嗎？

**A**: 可以！在建立/修改任務時，隨時可以切換模式。但切換時：
- 切換到範本模式：手動輸入的內容會被清空
- 切換到手動模式：已選擇的範本資料會保留在 invokeTarget 中

### Q5: 如何修改現有任務？

**A**: 
1. 點選任務列表中的「修改」按鈕
2. 系統會以手動模式載入資料（保留原始設定）
3. 您可以：
   - 繼續使用手動模式修改
   - 或切換到範本模式重新選擇

## 📋 Cron 表達式參考

### 常用表達式

| 表達式 | 說明 |
|-------|------|
| `0 0 1 * * ?` | 每天凌晨 1 點 |
| `0 0 */2 * * ?` | 每 2 小時 |
| `0 */30 * * * ?` | 每 30 分鐘 |
| `0 0 9,12,15 * * ?` | 每天 9、12、15 點 |
| `0 0 9-17 * * ?` | 每天 9-17 點的整點 |
| `0 0 0 * * MON` | 每週一凌晨 |
| `0 0 0 1 * ?` | 每月 1 號凌晨 |

### 格式說明

```
* * * * * ?
│ │ │ │ │ │
│ │ │ │ │ └─ 星期 (0-7, SUN-SAT)
│ │ │ │ └─── 月份 (1-12, JAN-DEC)
│ │ │ └───── 日期 (1-31)
│ │ └─────── 時 (0-23)
│ └───────── 分 (0-59)
└─────────── 秒 (0-59)
```

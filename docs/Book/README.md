## 🎯 核心功能

### ISBN 查詢
採用三層備援機制，確保資料準確性：

1. **第一層：台灣主站** (isbn.tw)
   - 優先搜尋台灣圖書資料
   - 資料完整度高

2. **第二層：NiceBooks 美國站** (us.nicebooks.com)
   - 涵蓋國際書籍
   - 圖片品質優良

3. **第三層：Google Books API**
   - 全球最大圖書資料庫
   - 支援多語言書籍

### 借閱管理
- **借閱申請** - 圖書借閱登記
- **審核機制** - 多層級審核流程
- **到期提醒** - 自動到期通知
- **續借功能** - 線上續借處理
- **歸還登記** - 圖書歸還記錄

### 圖書資訊
- **書籍資料** - 完整圖書詳細資訊
- **封面圖片** - 自動取得書籍封面
- **分類管理** - 圖書分類與標籤
- **庫存狀態** - 即時庫存查詢

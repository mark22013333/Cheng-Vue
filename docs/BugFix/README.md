## 🔍 問題排查流程

1. **重現問題** - 確認問題可重現
2. **定位原因** - 查看日誌、除錯程式碼
3. **修復問題** - 實施解決方案
4. **測試驗證** - 驗證修復效果
5. **文件記錄** - 記錄修復過程

## 📝 Bug 報告格式

建議包含以下資訊：
- 問題描述
- 重現步驟
- 預期行為
- 實際行為
- 環境資訊
- 錯誤日誌
- 解決方案
